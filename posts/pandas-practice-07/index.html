DataFrame indexing

User Guide

API reference

.loc()

import pandas as pd

df = pd.DataFrame(
    [
        [1, 3],
        [2, 4],
        [3, 5]
    ],
    index=['cobra', 'viper', 'sidewinder'],
    columns=['max_speed', 'shield']
)

print(df)
            max_speed  shield
cobra               1       3
viper               2       4
sidewinder          3       5

single label: returns row as a Series

res = df.loc['viper']
print(type(res))
print(res)
<class 'pandas.core.series.Series'>
max_speed    2
shield       4
Name: viper, dtype: int64

list of labels: returns a DataFrame

res = df.loc[['viper', 'cobra']]
print(type(res))
print(res)
<class 'pandas.core.frame.DataFrame'>
       max_speed  shield
viper          2       4
cobra          1       3

single lable for row and column: returns a single value

res = df.loc['viper','max_speed']
print(type(res))
print(res)
<class 'numpy.int64'>
2

labels for row and single label for column

res = df.loc['viper':'sidewinder', 'max_speed']
print(type(res))
print(res)
<class 'pandas.core.series.Series'>
viper         2
sidewinder    3
Name: max_speed, dtype: int64

Boolean list with the same length as the row axis

res = df.loc[[True, False, True]]
print(type(res))
print(res)
<class 'pandas.core.frame.DataFrame'>
            max_speed  shield
cobra               1       3
sidewinder          3       5

.iloc(): purely integer-location based indexing

a single integer

res = df.iloc[0]
print(type(res))
print(res)
<class 'pandas.core.series.Series'>
max_speed    1
shield       3
Name: cobra, dtype: int64

a list of integers

res = df.iloc[[1,2]]
print(type(res))
print(res)
<class 'pandas.core.frame.DataFrame'>
            max_speed  shield
viper               2       4
sidewinder          3       5

indexing row and column

res = df.iloc[0,1]
print(type(res))
print(res)
<class 'numpy.int64'>
3

: for the entire axis

res = df.iloc[:, :2]
print(res)
print(type(res))
            max_speed  shield
cobra               1       3
viper               2       4
sidewinder          3       5
<class 'pandas.core.frame.DataFrame'>
